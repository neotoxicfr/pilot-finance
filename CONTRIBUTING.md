# Contribuer à Pilot Finance

Merci de votre intérêt pour le projet ! Comme il s'agit d'un projet personnel, voici la marche à suivre :

1. **Ouvrir une Issue** : Discutez du changement que vous souhaitez effectuer avant de coder.
2. **Pull Request** : Soumettez vos modifications sur une branche séparée. Assurez-vous que le build Docker passe.
3. **Standards de Code** : Respectez les patterns TypeScript et React déjà en place.
4. **Sécurité** : Ne poussez jamais de fichiers `.env` ou de bases de données.
